### Regras de Negócio (RN)
**RN01:** O sistema deve impedir o registro de público para sessões cuja data e hora já tenham sido ultrapassadas.

**RN02:** Cada cinema deve ter um endereço completo único, incluindo cidade e estado, para permitir a filtragem por localidade.

**RN03:** O cancelamento de uma sessão só é permitido se não houver público registrado (ingressos vendidos) para a mesma.
